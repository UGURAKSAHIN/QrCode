# QR Code Generator

A Windows Forms application that generates random QR codes dynamically.  
The QR codes can contain random text, numbers, or URLs.

## Features

- Generate random QR codes
- Save QR codes as images such as PNG or JPG
- Copy QR codes to clipboard
- Change QR code size and image format
- Simple Windows Forms interface

## Technologies

- C#
- Windows Forms
- .NET Framework
- QRCoder

## Installation

1. Install .NET Framework.
2. Download or clone the project.
3. Open the project in Visual Studio.
4. Restore NuGet packages.
5. Run the application.

## Usage

1. Click the **Generate QR Code** button.
2. View the generated QR code.
3. Save or copy the QR code as needed.

## Dependencies

This project uses:

- [QRCoder](https://github.com/codebude/QRCoder)
